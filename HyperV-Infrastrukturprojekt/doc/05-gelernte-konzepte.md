# Gelernte Konzepte

Im Verlauf des Projekts wurden folgende Technologien und Konzepte erarbeitet:

- **Hyper-V:** Einrichtung und Verwaltung virtueller Maschinen
- **pfSense:** Konfiguration als Router/Firewall inkl. VLAN-Terminierung und Regelwerk
- **Virtuelle Switches:** Erstellen interner und externer vSwitches in Hyper-V
- **Domain Controller:** Installation und Konfiguration von Active Directory
- **DHCP / DNS:** Einrichten von Scopes und Zonen unter Windows Server
- **vNICs via PowerShell:** Trunk Ports / VLAN-Tags an virtuellen Netzwerkadaptern setzen
- **Firewall-Regeln:** Regelbasierte Zugriffskontrolle zwischen VLANs
- **DHCP-Relay:** Konfiguration eines DHCP-Relay unter pfSense
- **PowerShell Netzwerk:** Netzwerkprofilwechsel (öffentlich → privat)
- **IPSec:** Erstellung und Aufbau einer IPSec-Verbindung
- **Zabbix (Ubuntu):** Installation und Grundkonfiguration des Monitoring-Servers
- **Zabbix Monitoring:** Einrichtung von Hosts, Items und Triggern
